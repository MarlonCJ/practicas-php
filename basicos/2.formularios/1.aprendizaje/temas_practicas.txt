------------------------------------------------
✅ PASO 2 — Recibir datos con PHP usando $_POST
------------------------------------------------

Ahora vamos a capturar lo que el usuario escribe en el formulario.

🔧 Reemplaza tu archivo formulario.php por esto:

<?php

if ($_SERVER["REQUEST_METHOD"] == "POST") {

    $nombre = $_POST["nombre"];
    $email = $_POST["email"];

    echo "<h3>Datos recibidos:</h3>";
    echo "Nombre: " . $nombre . "<br>";
    echo "Email: " . $email . "<br>";
}

?>

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Formulario PHP</title>
</head>
<body>

<h2>Registro Usuario</h2>

<form action="" method="POST">

    Nombre:
    <input type="text" name="nombre"><br><br>

    Email:
    <input type="email" name="email"><br><br>

    <input type="submit" value="Enviar">

</form>

</body>
</html>

🎯 Qué hace este código

✅ 1. Detecta si se envió el formulario

$_SERVER["REQUEST_METHOD"] == "POST"

Significa:

👉 Si el usuario presionó Enviar

✅ 2. Captura datos

$_POST["nombre"]
$_POST["email"]

PHP toma lo escrito en los inputs.

✅ 3. Muestra resultado

echo

Imprime los datos en pantalla.

Email: correo@gmail.com

Y presiona enviar.

🔥 Aprendiste:

✅ $_POST
✅ Detectar envío
✅ Leer inputs
✅ Mostrar resultados

-------------------------------------------------------------------

Esto es muy importante en formularios reales.

👉 Si el usuario se equivoca, no pierde lo que escribió.

🔧 Reemplaza tu archivo por este código:

<?php

$error = "";
$nombre = "";
$email = "";

if ($_SERVER["REQUEST_METHOD"] == "POST") {

    $nombre = $_POST["nombre"];
    $email = $_POST["email"];

    if (empty($nombre) || empty($email)) {
        $error = "Todos los campos son obligatorios.";
    } else {
        echo "<h3>Datos correctos:</h3>";
        echo "Nombre: $nombre <br>";
        echo "Email: $email <br>";
    }
}

?>

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Formulario PHP</title>
</head>
<body>

<h2>Registro Usuario</h2>

<?php
if ($error != "") {
    echo "<p style='color:red;'>$error</p>";
}
?>

<form action="" method="POST">

    Nombre:
    <input type="text" name="nombre" value="<?php echo $nombre; ?>"><br><br>

</body>
</html>

🎯 Qué cambió

Antes:

Si fallaba → todo quedaba vacío.

Ahora:

Si fallaba → conserva lo escrito.

🔥 La clave está aquí:

Y:

value="<?php echo $email; ?>"

Eso rellena nuevamente los inputs.

---------------------------------------------------
✅ PASO 5 — Sanitizar datos + prevenir ataques XSS
---------------------------------------------------


Ahora entraremos en seguridad real.

Si alguien escribe código malicioso en un formulario, debemos bloquearlo.

⚠️ Ejemplo de ataque

Si alguien escribe en nombre:

<script>alert('hack')</script>

Sin protección, podría ejecutarse.

✅ Solución profesional: htmlspecialchars()

Convierte caracteres peligrosos en texto seguro.

🔧 Reemplaza tu archivo por este código:

<?php

$error = "";
$nombre = "";
$email = "";

if ($_SERVER["REQUEST_METHOD"] == "POST") {

    $nombre = htmlspecialchars($_POST["nombre"]);
    $email = htmlspecialchars($_POST["email"]);

    if (empty($nombre) || empty($email)) {
        $error = "Todos los campos son obligatorios.";
    } else {
        echo "<h3>Datos seguros recibidos:</h3>";
        echo "Nombre: $nombre <br>";
        echo "Email: $email <br>";
    }
}

?>

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Formulario PHP Seguro</title>
</head>
<body>

<h2>Registro Usuario</h2>

<?php
if ($error != "") {
    echo "<p style='color:red;'>$error</p>";
}
?>

<form action="" method="POST">

    Nombre:
    <input type="text" name="nombre" value="<?php echo $nombre; ?>"><br><br>

    Email:
    <input type="email" name="email" value="<?php echo $email; ?>"><br><br>

    <input type="submit" value="Enviar">

</form>

</body>
</html>

🎯 Qué hace esto

htmlspecialchars()

Convierte:

Entrada	Resultado

<	&lt;
>	&gt;
"	&quot;

📌 Prueba real

Escribe en nombre:

<script>alert('X')</script>

👉 Ahora se mostrará como texto, no se ejecuta.

✅ PASO 6 — Validar email correctamente con filter_var()
---------------------------------------------------------

Ahora vamos a validar si el correo realmente tiene formato válido.

No basta con que no esté vacío.

❌ Correos inválidos

juan
juan@
correo.com
@correo.com

✅ Correo válido

juan@gmail.com
admin@empresa.co

🔧 Reemplaza tu archivo por este código:

<?php

$error = "";
$nombre = "";
$email = "";

if ($_SERVER["REQUEST_METHOD"] == "POST") {

    $nombre = htmlspecialchars($_POST["nombre"]);
    $email = htmlspecialchars($_POST["email"]);

    if (empty($nombre) || empty($email)) {

        $error = "Todos los campos son obligatorios.";

    } elseif (!filter_var($email, FILTER_VALIDATE_EMAIL)) {

        $error = "Correo electrónico inválido.";

    } else {

        echo "<h3>Formulario enviado correctamente</h3>";
        echo "Nombre: $nombre <br>";
        echo "Email: $email <br>";
    }
}

?>

<!DOCTYPE html>
<html lang="es">
<head>
<meta charset="UTF-8">
<title>Formulario Seguro</title>
</head>
<body>

<h2>Registro Usuario</h2>

<?php
if ($error != "") {
    echo "<p style='color:red;'>$error</p>";
}
?>

<form method="POST">

Nombre:
<input type="text" name="nombre" value="<?php echo $nombre; ?>"><br><br>

Email:
<input type="text" name="email" value="<?php echo $email; ?>"><br><br>

<input type="submit" value="Enviar">

</form>

</body>
</html>

🎯 Qué hace esta línea

filter_var($email, FILTER_VALIDATE_EMAIL)

PHP revisa si el email tiene estructura válida.

Si no sirve:

false

📌 Haz pruebas

❌ Caso malo:
juan@

Debe salir:

Correo electrónico inválido

✅ Caso bueno:

juan@gmail.com

Debe aceptar.

✅ PASO 8 — Formulario profesional final con diseño + validación total
-----------------------------------------------------------------------

Ahora haremos una versión más realista: ordenada, con estilos simples, validación y mensajes claros.

🔧 Reemplaza tu archivo por este código:

<?php

$error = "";
$exito = "";

$nombre = "";
$email = "";
$ciudad = "";
$genero = "";
$hobbies = [];

if ($_SERVER["REQUEST_METHOD"] == "POST") {

    $nombre = htmlspecialchars(trim($_POST["nombre"] ?? ""));
    $email = htmlspecialchars(trim($_POST["email"] ?? ""));
    $ciudad = $_POST["ciudad"] ?? "";
    $genero = $_POST["genero"] ?? "";
    $hobbies = $_POST["hobbies"] ?? [];

    if ($nombre == "" || $email == "") {
        $error = "Nombre y correo son obligatorios.";
    } elseif (!filter_var($email, FILTER_VALIDATE_EMAIL)) {
        $error = "Correo inválido.";
    } elseif ($ciudad == "") {
        $error = "Debes seleccionar una ciudad.";
    } elseif ($genero == "") {
        $error = "Debes elegir género.";
    } else {
        $exito = "Formulario enviado correctamente.";
    }
}

?>

<!DOCTYPE html>
<html lang="es">
<head>
<meta charset="UTF-8">
<title>Formulario Profesional</title>

<style>
body{
    font-family: Arial;
    background:#f4f4f4;
}

    width:420px;
    background:white;
    padding:25px;
    margin:40px auto;
    border-radius:10px;
    box-shadow:0 0 15px rgba(0,0,0,.15);
}

input, select{
    width:100%;
    padding:10px;
    margin-top:5px;
    margin-bottom:15px;
}

button{
    width:100%;
    padding:12px;
    background:#007bff;
    color:white;
    border:none;
    cursor:pointer;
}

.error{
    color:red;
    margin-bottom:15px;
}

.ok{
    color:green;
    margin-bottom:15px;
}
</style>

</head>
<body>

<div class="contenedor">

<h2>Registro Profesional</h2>

<?php if($error != "") echo "<div class='error'>$error</div>"; ?>
<?php if($exito != "") echo "<div class='ok'>$exito</div>"; ?>
